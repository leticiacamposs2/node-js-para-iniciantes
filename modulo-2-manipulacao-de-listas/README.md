# Módulo 02 - Manipulação de listas 📜

Neste módulo é feito o entendimento de manipulação de listas.

For, Foreach, ForIn, Map, Filter, Reduce

=> [🍔, 🍟, 🍗, 🍿]

filter([🍔, 🍟, 🍗, 🍿], isVegetarian)
=> [🍟, 🍿]

reduce([🍔, 🍟, 🍗, 🍿], eat)
=> 💩
